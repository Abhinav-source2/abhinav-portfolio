## 📂 Project Structure
```
AJ_PORTFOLIO_full/
├── public/                    # Public assets (Resume, images)
├── src/
│   ├── assets/                # Static images & tech icons
│   ├── components/            # Reusable components (Hero, Navbar, etc.)
│   ├── App.jsx                # Main routing file
│   ├── index.css              # Global Tailwind styles
│   └── main.jsx               # Entry point
├── tailwind.config.js        # Tailwind theme settings
├── vite.config.js            # Vite bundler setup
└── README.md                 # You're here!
```

## 🧠 Features

- ⚡ **Fully Responsive** with TailwindCSS
- 🎨 **Dark Mode Toggle**
- 💬 **Animated Typing Text**
- 🚀 **Framer Motion Animations**
- 🧠 **Projects with Tech Stack Tags & GitHub Links**
- 📄 **Resume Download (Hosted from /public)**
- 💡 **Contact Form (Formspree Integrated)**
- 🌐 **Deployed on Vercel**

## 📥 Installation

```bash
git clone https://github.com/Abhinav-source2/portfolio.git
cd portfolio
npm install
npm run dev
```

---

## 🧾 Customization
- 🔧 Replace `public/Abhinav_Jajoo_Resume.pdf` with your own.
- 🎯 Edit `src/components/Projects.jsx` to add your projects.
- 🎨 Modify `tailwind.config.js` to adjust fonts/colors.
